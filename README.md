# inktty

A simple terminal markdown reader built with OpenTUI.

## Usage

```bash
# Read a markdown file
bun run bin/ink.tsx <path-to-file.md>

# Development (watch mode)
bun run dev

# Run tests
bun test
bun run test:bdd

# Lint and format
bun run lint
bun run lint:fix
```

## Example

```bash
bun run bin/ink.tsx example.md
```

Press **q** to quit.
